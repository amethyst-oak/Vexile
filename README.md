**Vexile** is an ultra-lightweight, high-speed, and fully open-source Minecraft launcher built for those tired of heavy Electron-based launchers that consume gigabytes of RAM. No unnecessary background bloat—just pure native Java Swing and maximum performance.

---

## ✨ Key Features

* **⚡ Feather-light:** No heavy web frameworks or bloated dependencies. Written in native Java Swing, ensuring instant startup and minimal resource consumption.
* **🔄 Retro-focused (Oldify Mode):** Perfect for launching classic Minecraft versions (Alpha, Beta, early releases) and custom community servers with your friends.
* **📥 Direct Manifests:** Dynamically fetches game versions, libraries, and assets straight from official Mojang servers.
* **🔓 Open-Source:** Fully open-source under the MIT License—inspect, modify, and build it your way.

---

## 🚀 How to Download & Compile

### For Users (Pre-built .JAR)
1. Go to the [Releases](../../releases) section of this repository.
2. Download the latest `VexileLauncher.jar`.
3. Make sure you have Java installed (supports Java 8 and higher).
4. Run the launcher via terminal or double-click:
   ```bash
   java -jar VexileLauncher.jar
   ```
### For Developers:
1. Clone the repository:
   ```bash
   git clone amethyst-oak/Vexile
   ```
2. Open the folder and run build.sh
   ```bash
   sh build.sh (if doesn't work, do chmod +x build.sh first)
   ```
3. Get the .jar in the dist folder.
